Using the CLI

In your terminal, activate the virtual environment and use the CLI:

# First, activate the virtual environment
source .venv/bin/activate

# Then ingest files
slidex ingest file /path/to/presentation.pptx

# Ingest a folder (recursive)
slidex ingest folder /path/to/folder --recursive

# Ingest a folder (non-recursive)
slidex ingest folder /path/to/folder --no-recursive

Using the API

You can also use the REST API directly:

# Ingest file via API
curl -X POST http://localhost:5000/api/ingest/file \
  -H "Content-Type: application/json" \
  -d '{"path": "/path/to/file.pptx"}'

# Ingest folder via API
curl -X POST http://localhost:5000/api/ingest/folder \
  -H "Content-Type: application/json" \
  -d '{"path": "/path/to/folder", "recursive": true}'

Using Python Directly

If you're in the project directory:

# Activate virtual environment
source .venv/bin/activate

# Use Python directly
python -c "from slidex.core.ingest import ingest_engine; ingest_engine.ingest_file('/path/to/file.pptx')"
